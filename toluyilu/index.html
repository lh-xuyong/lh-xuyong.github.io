大灰狼致小白兔

· 2023-4-15 · 次阅读

与你相遇，人生之幸

致我曾经最爱的人

    你好小白兔，当你转账过来时我的大脑一片空白，而后意识到我们最终还是失去了彼此。我的脑海中无数次构建有你的未来，想着我们的幸福，想着人生路上必定会遇到的困苦，想着如何为你披荆斩棘。然而现实却是如此的荒诞。你给的第一次牵手、第一个拥抱以及白马湖大桥上的吻，这些都是我生命中不可磨灭的印记，感谢你曾经的爱。这些天的断联让回归理性的我意识到自己的恋爱心态出了问题，那时的我已经喜欢你到了失去了智商变得卑微，变得患得患失，慢慢丢掉了自尊，没了自信，丧失了自己本该有的魅力,不再优雅。我会积极的健身、努力学习专业知识，做出积极的改变认真经营自己的下一段感情，也祝你早日找到属于自己的白马王子，携手共进。也许若干年后的一天我们会相遇于栖霞小区：相视一笑，互道幸福美满。

为你写的死循环终究还是停了下来

2023年4月7日终止循环

public static void main(String[] args) throws ParseException {
    /开始的日子
    String loveDate = "2023-2-19 10:11:00";
    //将本天和19号转为时间戳
    SimpleDateFormat dateFormat = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
    Long loveDateToU = dateFormat.parse(loveDate).getTime();
    Long day =  System.currentTimeMillis();
    //开始死循环,只要时间大于19号就输出并+1天
    for(;loveDateToU < day;day=day+86400000){
        Date date = new Date(loveDateToU);
        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
        String formattedDate = sdf.format(date);
        try {
            Thread.sleep(5000); // 睡一会儿
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

        System.out.println(""+formattedDate+"想你");
        //到了2023年4月7日终止循环
        if(day = "2023-04-07"){
            System.out.println(""+formattedDate+"终止循环");
            break;
        }
        loveDateToU += 86400000;
    }
}

难以诉说的心情以视频代替

没有人能够持续爱一个人，真正的爱情，是在漫长的生活中一次次重新爱上对方。

  1. 与你相遇,人生之幸。
  2. 时间是人最伟大的导师，我见过无数被失恋折磨的死去活来的人，是时间帮助他们抚平了心灵的创伤。
  3. 村上春树说：感情是两个人一起做出改变，相互融合习惯,千万别相信自己能多幸运，往往过多奢求，最是让你失落至极。说绝望太过了，说伤心好像也不太贴切，到不如是失落，有过幻想才会失落，所以不要太过于把目光投注于别人身上。
