五、重定向

Shell对于每个进程预先定义了3个文件描述字

0 标准输入 STDIN 键盘

1 标准输出 STDOUT 显示器

2 标准错误输出 STDERR

重定向，就是改变这个标准设备，不用键盘输入，不用显示器输出

符号 作用
命令 < 文件 将文件作为命令的标准输入
命令 << 文件 从标准输入中读入，用分界符起止
命令 > 文件 将标准输出输入到一个文件中
命令 >> 文件 将标准输出输入到一个文件中（追加）
命令 2> 文件 将错误输出重定向到文件中
命令 2>> 文件 将错误输出重定向到文件中（追加）

符号 作用

命令 | | 命令 或运算
