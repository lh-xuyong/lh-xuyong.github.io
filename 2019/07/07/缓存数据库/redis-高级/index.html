相关命令

键值相关命令
keys 键名 按照键名查找指定的键。支持通配符
exists 键名 确认一个键是否存在
del 键名 删除一个键
expire 键 秒 设置一个键的过期时间，如果键已经过期，将会被自动删除
ttl 键 以秒为单位，返回键的剩余生存时间
select 数据库号 选择一个数据库。 默认连接的数据库是0，可以支持共16个数据库 . 在配置文件中，通过databases 16 关键字定义
move 键 数据库号 将当前数据库的键移动到指定的数据空中
randomkey 从当前数据库返回一个随机的键。如果当前库没有任何键，则返回nil
type 键 返回键类型 html> ragment–>
rename 旧名 新名 重命名键
服务器相关命令
ping 测试服务器是否可以连接
echo 字符串 在命令行输出字符串
quit 退出redis数据库
save 保存所有的数据。很少在生产环境直接使用SAVE 命令，因为它会阻塞所有的客户端的请求，可以使用BGSAVE 命令代替. 如果在BGSAVE命令的保存数据的子进程发生错误的时,用 SAVE命令保存最新的数据是最后的手段
dbsize 返回当前库中键的数量
info 获取服务器的详细信息
config get 参数 获取redis服务器配置文件中的参数。支持通配符
flushdb 删除当前数据库中所有的数据
flushall 删除所有数据库中所有的数据
