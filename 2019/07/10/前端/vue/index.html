学习未来的前端开发巨头——Vue.js

MVVM

MVVM是前端视图层的概念,主要关注于视图层分离,也就是说:MVVM把前端的视图层,分为了三部分 MOdel, View, VM Viewmodel

  • M:保存的是每个页面中单独的数据(通过ajax从服务器拿到一个数组,这个数组即M)
  • VM:调度者,分割了M和V,每当V想要获取后台数据,都要由VM做中间处理
  • V:就是每个页面中的html

MVVM思想作用:
是为了让我们开发更加方便,因为MVVM提供了数据的双向绑定,注意:数据的双向绑定是由VM提供的
