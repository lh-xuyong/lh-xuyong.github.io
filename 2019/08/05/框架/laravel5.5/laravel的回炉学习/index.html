路由

基础

  • 与TP不同,Laravel占位符用的是{}
  • 可以用于where对后面参数做正则限制,例如只能是数字.
  • id后面加?表示可选参数

别名

路由组

  • []内不写规则,则路由组内路由正常使用(只是用于分类)

  • [中间件]就代表着路由组内的每条路由,都必须登录过后才能访问

路由资源控制器

请求方式 URI路径 控制器方法 路由名称
GET /order index (列表) order.index
GET /order/create create (添加页面) order.create
POST /order store (添加逻辑) order.store
GET /order/{id} show (查看某一条详情) order.show
GET /order/{id}/edit edit (修改某条页面) order.edit
PUT/PATCH /order/{id} update order.update
DELETE /order/{id} destroy order.destroy

常见请求

获取基本信息

获取基本信息
$request->method() 获取请求方法
$request->isMethod(‘post’) 检测方法
$reques->path() 请求路径
$request->url() 获取完整url
$request->ip() 获取ip
$request->getPort() 获取端口

获取参数

获取参数
$name = $request->input(‘name’); 基本获取
$request->input(‘name’, ‘Sally’); 设置默认值
$request->has(‘name’) 检测是否存在
$input = $request->all(); 提取所有的参数
$request->only([‘username’, ‘password’]); 只要这部分
$request->except([‘credit_card’]); 不要这部分
$request->header(‘Connection’) 获取头信息

文件操作

文件操作
$request->hasFile(‘photo’) 检测是否有文件上传
$request->file(‘photo’)->move(‘./uploads/‘, ‘iloveyou.jpg’); 将文件移动到指定位置

Cookie

Cookie
\Cookie::queue(‘name’,’xuyong’,10); 设置(第三个参数以分钟为单位)
\Cookie::get(‘name’); 读取
$request->cookie(‘name’); 读取

Session

Session
session([‘Mobile_user_name’=> $p[‘name’]]); 设置session
$request->session()->put(‘name’,$userName); 设置session
$request->session()->flush(); 删除session
闪存
即只可以使用一次的session
\Session::flash(‘name’,’ayong’,10) 设置
\Session:get(‘name’) 获取
\Session:has(‘name’) 判断

响应

response
response()->json([‘a’=>100,’b’=>2000]); 返回json
json.parse(‘{“name”:”xuyong”,”age”:22}’)或$.parseJSON json转成JS对象
json_decode(‘{“name”:”xuyong”,”age”:22}’,true) 加true,json转数组.不加转对象
redirect (‘task/task’) 页面跳转
response()->view(‘goods.add’); 显示模板
response()->download(‘../readme.md’) 返回下载(放一堆链接,用户一点即是下载)
